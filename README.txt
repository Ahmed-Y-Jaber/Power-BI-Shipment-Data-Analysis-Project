# Shipment Data Analysis Project

This project analyzes shipment data using Power BI, focusing on key logistics metrics like delivery times and revenue.

## Features
- **Data Cleaning**: Imported and cleaned shipment data.
- **Visualizations**: Created interactive visuals to track shipment performance and analyze delivery times.
- **Insights**: Generated actionable insights to empower decision-makers in logistics operations.

## Key Visuals:
- Shipment performance overview
- Delivery times analysis
- Revenue tracking

### Tips & Tricks for Power BI:
- Use of slicers to enhance interactivity
- Custom themes for a professional look
- Conditional formatting to highlight key metrics
